# 幼乐码 - 面向幼儿园的工业级STEAM编程启蒙解决方案

## 1. 概念与愿景

幼乐码是一款专为3-6岁幼儿设计的STEAM编程启蒙平台，将工业级编程逻辑通过游戏化、可视化的方式呈现，让幼儿在"玩中学、学中玩"的过程中培养计算思维和创新能力。界面采用明亮活泼的视觉风格，配合大图标、简单手势和即时反馈机制，完美适配幼儿的认知特点。

## 2. 设计语言

### 2.1 美学方向
**童趣科技风** - 结合可爱卡通元素与科技感，呈现"友好的未来世界"氛围。圆润的边角、柔和的阴影、大尺寸的交互元素，确保幼儿能够轻松操作。

### 2.2 色彩系统
```css
:root {
  /* 主色调 */
  --primary: #FF6B6B;        /* 珊瑚红 - 主按钮、重点强调 */
  --primary-light: #FFE5E5;  /* 浅珊瑚 - 背景装饰 */

  /* 次要色 */
  --secondary: #4ECDC4;      /* 薄荷绿 - 成功状态、正向反馈 */
  --secondary-light: #E0F7F5;

  /* 点缀色 */
  --accent-yellow: #FFE66D;  /* 阳光黄 - 成就、奖励、星星 */
  --accent-blue: #45B7D1;    /* 天空蓝 - 信息、学习进度 */
  --accent-purple: #A78BFA; /* 梦幻紫 - 创意、特殊功能 */

  /* 中性色 */
  --bg-primary: #FFF9F0;    /* 奶油白 - 主背景 */
  --bg-secondary: #FFFFFF;   /* 纯白 - 卡片背景 */
  --bg-dark: #2D3436;        /* 深灰 - 编程区域背景 */

  /* 文字色 */
  --text-primary: #2D3436;  /* 主文字 */
  --text-secondary: #636E72;/* 次要文字 */
  --text-light: #FFFFFF;     /* 深色背景上的文字 */
}
```

### 2.3 字体系统
- **标题字体**: "Nunito" (Google Fonts) - 圆润友好，适合幼儿界面
- **正文字体**: "Noto Sans SC" - 中文显示清晰
- **编程字体**: "Fira Code" - 代码块展示
- **字号规范**: 最小正文字号 16px，标题最小 24px，按钮文字最小 18px

### 2.4 空间系统
- 基础单位: 8px
- 间距层级: 8px / 16px / 24px / 32px / 48px
- 圆角: 小元素 8px / 卡片 16px / 大容器 24px
- 触摸目标: 最小 48px × 48px（幼儿友好）

### 2.5 动效哲学
- **即时反馈**: 所有交互都有视觉或音效反馈
- **弹性动画**: 使用 spring 缓动，传达活泼感
- **进度展示**: 加载时显示趣味动画（如小火车、星星）
- **成就动画**: 完成挑战时的庆祝效果（星星飞溅、彩带）
- 动画时长: 微交互 150ms / 页面切换 300ms / 复杂动画 500ms

### 2.6 视觉资产
- **图标库**: Lucide Icons (圆润风格)
- **吉祥物**: 可爱的机器人"乐乐"作为引导角色
- **背景纹理**: 淡淡的云朵、星星、圆形几何图案
- **插图风格**: 扁平化卡通风格，色彩饱和度高

## 3. 布局与结构


### 3.2 响应式策略
- **平板优先** (768px-1024px): 主要适配幼儿园教学平板

- **移动端** (320px-768px): 家长端功能

## 4. 功能与交互

### 4.1 幼儿端核心功能

#### 4.1.1 角色选择首页
- 大图标展示三个角色入口：学生 / 教师 / 管理员
- 点击进入对应登录页
- 显示吉祥物"乐乐"欢迎动画

#### 4.1.2 可视化编程工作区
**积木式编程界面**
- 左侧: 角色展示区（舞台）- 80% 宽度
- 右侧: 积木工具箱 - 20% 宽度
- 底部: 代码序列区（已组装的积木）

**积木分类**
1. 运动积木: 前进、后退、左转、右转、跳
2. 外观积木: 说Hello、换装、变大/小
3. 声音积木: 播放音效、背景音乐
4. 控制积木: 重复N次、如果那么
5. 事件积木: 当开始、碰撞检测

**交互行为**
- 拖拽积木到代码区，自动吸附
- 点击积木高亮，可删除或重新排序
- "运行"按钮执行代码序列
- "重置"按钮恢复初始状态
- 完成挑战后显示星星奖励

#### 4.1.3 游戏化课程
- 课程卡片展示：图片 + 标题 + 进度条 + 星星数
- 课程分类：小班基础 / 中班进阶 / 大班拓展
- 闯关模式：每节课一个小任务，完成解锁下一关
- 成就系统：收集星星、徽章、称号

### 4.2 教师端核心功能

#### 4.2.1 教学概览
- 今日课程安排表
- 学生出勤情况
- 班级整体学习进度
- 最近学生作品展示

#### 4.2.2 课程管理
- 课程列表: 名称、年级、时长、难度
- 课程编辑器: 添加/编辑课程内容
- 作业布置: 选择课程发布为作业
- 课程预览: 模拟学生视角查看

#### 4.2.3 学生管理
- 学生列表: 头像、姓名、班级、积分
- 学习报告: 单个学生的学习轨迹
- 分组管理: 创建学习小组
- 批量导入/导出

#### 4.2.4 学习分析
- 班级平均进度图表
- 学生能力雷达图
- 课程完成率统计
- 热力图: 学习时间段分析

### 4.3 管理端核心功能

#### 4.3.1 数据看板
- 核心指标: 总用户数、今日活跃、课程数、完成率
- 趋势图表: 周/月用户增长曲线
- 实时动态: 最新注册、课程完成动态
- 地图可视化: 幼儿园分布

#### 4.3.2 课程管理
- 课程库: 所有课程统一管理
- 课程审核: 教师提交课程的审批
- 课程推荐: 首页推荐位管理
- 课程分类: 年级/科目/难度

#### 4.3.3 用户管理
- 用户列表: 分角色查看
- 权限管理: 角色权限配置
- 机构管理: 幼儿园/班级层级
- 数据导出: 用户数据报表

## 5. 组件清单

### 5.1 通用组件

#### Button 按钮
- **变体**: Primary / Secondary / Ghost / Outline
- **尺寸**: Small (32px) / Medium (44px) / Large (56px)
- **状态**: Default / Hover (scale 1.05) / Active (scale 0.98) / Disabled (opacity 0.5) / Loading (显示旋转图标)
- **幼儿版**: 更大尺寸(64px)、圆角(24px)、阴影效果

#### Card 卡片
- 白色背景、16px圆角、轻微阴影
- Hover: 轻微上浮 + 阴影加深
- 选中状态: 边框高亮

#### Progress 进度条
- 圆角条形、渐变填充
- 动画: 从0到当前值平滑过渡
- 显示百分比和星星数量

#### Modal 对话框
- 居中显示、背投模糊
- 标题 + 内容 + 操作按钮
- 关闭: 点击X或背景

#### Input 输入框
- 大尺寸(48px高度)、圆角(12px)
- Label 始终显示
- 错误状态: 红色边框 + 错误提示

### 5.2 业务组件

#### CodeBlock 编程积木
- 彩色积木块、带把手图标
- 可拖拽、圆角边
- 连接线表示组合关系

#### Stage 舞台区
- 方形或圆形舞台
- 角色精灵居中显示
- 背景可切换

#### AchievementBadge 成就徽章
- 圆形徽章 + 图标
- 锁定状态: 灰色遮罩
- 获得动画: 缩放 + 发光

#### CourseCard 课程卡片
- 封面图 + 标题 + 描述
- 进度条 + 已获星星
- 状态标签: 必修/选修/已完成

#### StudentAvatar 学生头像
- 圆形头像 + 等级标签
- 在线状态指示
- Hover 显示详细信息

#### StatCard 统计卡片
- 图标 + 数字 + 标签
- 趋势指示(上升/下降箭头)
- 颜色编码区分类型

## 6. 技术方案

### 6.1 技术栈
- **框架**: Next.js 16 (App Router)
- **语言**: TypeScript 5
- **UI组件**: shadcn/ui (基于 Radix UI)
- **样式**: Tailwind CSS 4
- **图标**: Lucide React
- **图表**: Recharts
- **状态管理**: React useState/useContext
- **数据模拟**: 内置 Mock Data

### 6.3 关键实现

#### 可视化编程引擎
- 基于 HTML5 Canvas 或 SVG 实现舞台渲染
- 积木拖拽使用 React DnD 或自定义手势处理
- 代码执行使用模拟器，实时反馈角色行为

#### 响应式适配
- Tailwind CSS 断点: sm(640px) / md(768px) / lg(1024px) / xl(1280px)
- 平板横竖屏适配
- 触摸事件优化

### 6.4 性能优化
- 图片懒加载
- 组件按需加载
- 动画使用 CSS transform/opacity
- 避免重排重绘
